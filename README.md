# Ambient Calculator
An open-sourced dynamic and beautiful calculator for Android phone and tablet. 

<div align="left">

## 📖 Features

* None permissions required
* Dark, Light, Material You and Google Material 3 theme
* Scientific mode
* History
* Optimised for phone (single panel) and tablet (dual panel)
* Quick settings tile
* Fast Speed

## ⚠️ Requirements

* Android 10.0+
